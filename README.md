# Ulga

This is a Hugo site initialized in `/Users/kubamazurkiewicz/apps/ulga`.

## Run locally

```bash
cd /Users/kubamazurkiewicz/apps/ulga
hugo server --buildDrafts
```

## Add content

- Add pages under `content/`
- Customize templates under `layouts/`
- Optionally add a theme under `themes/` and set `theme = "<your-theme>"` in `hugo.toml`
